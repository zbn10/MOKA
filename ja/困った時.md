<a id="freeze"></a>
## 突然反応しなくなった

<a id="invalidkeymap"></a>
## 使っている最中にキーマップがおかしくなった
予期せず別のキーマップを読み込む機能を起動した可能性があります。目的のキーマップを読み込んでみてください。  
予期せずOSLayout切替を押した可能性があります。OSLayout切り替えを押して動作を再度確認してみてください。

<a id="invalidswresponse"></a>
## 特定の磁気スイッチの反応が悪い、入力できない、あるいはエラー表示される
キーを押した状態でキーボードを起動すると、磁気スイッチのリリース位置判定がおかしくなり、そのキーを押せなくなることがあります。キーを押していない状態でキーボードをリセットしてみてください。
キースイッチを交換したり環境がかわった場合、磁気スイッチのリリース位置・押し込み位置のセンサー値が変わり、正しくキー状態が読み取れなくなります。キャリブレーションを実施してください。
それでも治らない場合はセンサー接続不良の可能性があります。該当キーのセンサーの接続をテスターなどで確認してみてください。

<a id="slowbehavior"></a>
## キー入力やタッチパッド動作が重くてモタつく

<a id=""></a>
## LEDやOLEDでMCUエラーが表示され、キー入力ができない
全てのキーが反応しない場合、磁気センサーを読み取るサブマイコンに電源が供給されていない可能性があります。
左手基盤の裏にある切り替えスイッチが UPDI 側になっていたら、NORM 側に切り替えてからメインマイコンをリセットしてください。

<a id="righterror"></a>
## 右側のキーやパッドが動かない
左右連携のケーブルが正しく接続されていない可能性があります。ケーブルを左右とも抜いて、もう一度接続してみてください。

<a id="misscalibration"></a>
## キャリブレーションしたのに反映されない
一部のキーにキャリブレーションの結果が正しく反映されない場合、キャリブレーション中にそのキーを押し忘れた可能性があります。もう一度キャリブレーションモードに入って、該当キーを何度か押してキャリブレーションモードを抜けてください。  
全てのキーにキャリブレーションの結果が正しく反映されない場合、キャリブレーションモードを抜ける操作をせずキーボードリセット等を行った可能性があります。もう一度キャリブレーションをやり直してください。結果はキャリブレーションモードを抜ける操作をする際に保存されます。

## MOKA って何の略？
よくある後付け略称です。  
Magnetic Optimized Keyboard Architecture  
Magnetic & Optical Keyboard Analog-actuated  
Magnetic Obsessive Key Addiction  
あたりがいいと思います。
