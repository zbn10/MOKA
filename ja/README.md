## 4. 機能説明
### 4.1 キーボード全体の説明

デバイス全体の説明。
表
裏

キーボードは NKRO 入力対応。

タッチパッドは windows precision touchpad として動作。
マルチタッチは、デバイスの構造上は3タッチ以上も可能だが、パット面積が狭いので複数タッチの位置判定の精度を出すのが難しく、ファームウェアで実装していない。
左右のパッドそれぞれ1タッチ、両方合わせて2タッチをサポート。

|機能|概要|
|:----|:----|
|アクチュエーションポイント設定|キー毎の設定が可能<br>0(リリース)～255(プレス)の範囲で指定|
|入力判定に使えるキーの各種状態|半押し<br>全押し<br>長押し1(1秒)<br>長押し2(3秒)|
|センサーキャリブレーション|42個の磁気センサーごとにスイッチ押し込み位置のセンサー値を学習<br>キーボードの使い始め、磁気スイッチを交換したときに実施|

## 6. 既知の問題

|問題|概要|ステータス|
|:----|:----|:----|
|操作が早すぎてたまにフリーズ|TinyUSBスタックに起因するレースコンディション問題。<br>HIDレポート頻度を落とすことでかなり緩和。<br>それでも運が悪いとたまにフリーズする。|完全には未解決<br>ワークアラウンド対処済み|
|タッチパッド表面が雑|サンドペーパースキルがまだ低いので受け入れてください|徐々に改善予定|
