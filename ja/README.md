# はじめに
本キーボード(MOKA)を使うための機能説明や操作方法を記載する。  


# 入力一覧
タイピング・キーボード動作のトリガーとなる入力一覧

|種類|数|概要|
|:---|:---|:---|
|磁気スイッチ|42|本キーボードのメイン入力|
|タクトスイッチ|1|補助的に利用|
|タッチパッド|2|左右をあわせて一つの横長仮想タッチパッドとして動作|
|人感センサー|1|いまのところマウスジグラー用途|
|光量センサー|1|いまのところLEDの明るさ自動調整用途|

# 磁気スイッチタイピング機能
本キーボードのメインである磁気スイッチに関わる機能。

|機能|概要|
|:----|:----|
|アクチュエーションポイント設定|キー毎の設定が可能<br>0(リリース)～255(プレス)の範囲で指定|
|入力判定に使えるキーの各種状態|半押し<br>全押し<br>長押し1(1秒)<br>長押し2(3秒)|
|センサーキャリブレーション|42個の磁気センサーごとにスイッチ押し込み位置のセンサー値を学習<br>キーボードの使い始め、磁気スイッチを交換したときに実施|

# キーボードの操作・メンテ機能一覧

|機能|概要|
|:----|:----|
|キーマップ編集||
|キーマップ保存・読出||
|ブラウザからのGUIキーマップ編集|調整中<br>機能はあるものの、マップ可能な動作一覧が多くブラウザ側GUIアプリ頓挫中<br>via 対応したい|
